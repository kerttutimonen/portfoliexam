Theme 01: Basic Web

Title description, Dec 7, 2017
Image

This task was to make a responsive website. Each of the students were given a theme, and the website was supposed to follow that specific style in terms of the display and content. The theme that was given to me was "blue note". The website had to be responsive, have at least two breakpoints in css, use css grid on the layout and have a valid css that was put into an external stylesheet.

The Process

Title description, Sep 2, 2017

The theme blue note was introduced to me already in previous projects (ADD LINK). I had already made a mood board and style sheet for the previous versions, but since along the way I had improved my skills, I wasn't fully satisfied with the looks and code anymore. I decided to change some things - although my website was still partly following the style I used to have.
